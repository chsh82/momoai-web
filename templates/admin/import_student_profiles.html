Excel 일괄 임포트

구글 설문 결과 Excel 파일을 업로드하여 여러 학생의 프로필을 한번에 생성합니다

📌 사용 방법

  1. 1. 구글 설문 결과를 Excel 파일(.xlsx 또는 .xls)로 다운로드합니다
  2. 2. Excel 파일의 "1. 학생의 이름" 컬럼에 있는 이름으로 시스템의 학생을 찾습니다
  3. 3. 시스템에 해당 학생이 존재하고 프로필이 없는 경우에만 생성됩니다
  4. 4. 이미 프로필이 있는 학생은 건너뜁니다
⚠️

주의사항

• Excel 파일의 학생 이름이 시스템에 등록된 학생 이름과 정확히 일치해야 합니다

• 이미 프로필이 있는 학생은 업데이트되지 않습니다 (수동 수정 필요)

지원 형식: .xlsx, .xls
