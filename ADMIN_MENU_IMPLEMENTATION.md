## 🔧 기술 구현

### 1. Alpine.js 통합
```html
<!-- CDN 추가 -->
<script defer src="https://cdn.jsdelivr.net/npm/@alpinejs/collapse@3.x.x/dist/cdn.min.js"></script>
<script defer src="https://cdn.jsdelivr.net/npm/alpinejs@3.x.x/dist/cdn.min.js"></script>
```

### 2. 상태 관리
```javascript
x-data="{
    academic: true,      // 학사 관리 (기본 열림)
    members: false,      // 회원 관리
    learning: false,     // 학습 관리
    zoom: false,         // 온라인 강의실
    communication: false,// 커뮤니케이션
    finance: false,      // 재무 관리
    content: false       // 콘텐츠 관리
}"
```

### 3. 접기/펼치기 애니메이션
```html
<button @click="academic = !academic">
    <svg :class="{ 'rotate-180': academic }">
        <!-- 화살표 아이콘 -->
    </svg>
</button>
<div x-show="academic" x-collapse>
    <!-- 하위 메뉴 -->
</div>
```

### 4. 색상 코드 시스템
- 🔵 **학사 관리**: blue (파란색)
- 🟢 **회원 관리**: green (초록색)
- 🟣 **학습 관리**: purple (보라색)
- 🔷 **온라인 강의실**: indigo (인디고)
- 🟡 **커뮤니케이션**: yellow (노란색)
- 🟠 **재무 관리**: orange (주황색)
- 💗 **콘텐츠 관리**: pink (분홍색)

### 5. 호버 효과
```css
.sidebar-link-sub:hover {
    transform: translateX(4px);  /* 오른쪽으로 살짝 이동 */
}
```

## 🎯 사용자 가이드

### 메뉴 사용법
1. **카테고리 클릭**: 하위 메뉴 열기/닫기
2. **하위 메뉴 클릭**: 해당 페이지로 이동
3. **현재 위치**: 파란색/색상 하이라이트로 표시

### 팁
- 자주 사용하는 카테고리는 열어둔 채로 사용 가능
- 여러 카테고리를 동시에 열 수 있음
- 하위 메뉴 호버 시 오른쪽으로 살짝 이동하는 효과
