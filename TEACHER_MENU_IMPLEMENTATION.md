### 1. 계층 구조 재구성
```
🏠 대시보드 (단일 메뉴)

  └─ 독서 논술 MBTI

✍️ 과제 관리 (2개 하위 메뉴)
  ├─ 과제 출제/첨삭
  └─ 첨삭 관리

📁 학습 자료 (단일 메뉴)

💬 소통 (4개 하위 메뉴)
  ├─ 공지사항
  ├─ 클래스 게시판
  ├─ 강사 게시판
  └─ 하크니스 게시판
```

**총계**: 6개 카테고리, 15개 하위 메뉴

## 🔧 기술 구현

### 1. Alpine.js 상태 관리
```javascript
x-data="{
    courses: true,        // 내 수업 (기본 열림)
    students: false,      // 학생 관리
    assignments: false,   // 과제 관리
    communication: false  // 소통
}"
```

### 2. 색상 코드 시스템
- 🔵 **내 수업**: blue (파란색)
- 🟢 **학생 관리**: green (초록색)
- 🟣 **과제 관리**: purple (보라색)
- 🟡 **소통**: yellow (노란색)

### 3. 카테고리 구조
```html
<button @click="courses = !courses"
        class="w-full flex items-center justify-between px-4 py-3 rounded-lg hover:bg-gray-50">
    <div class="flex items-center gap-3">
        <span class="text-xl">📚</span>
        <span class="font-medium text-gray-700">내 수업</span>
    </div>
    <svg :class="{ 'rotate-180': courses }"
         class="w-5 h-5 text-gray-400 transform transition-transform">
        <!-- 화살표 아이콘 -->
    </svg>
</button>
<div x-show="courses" x-collapse>
    <a href="{{ url_for('teacher.courses') }}"
       class="sidebar-link-sub hover:text-blue-600">
        수업 목록
    </a>
    <!-- 더 많은 하위 메뉴... -->
</div>
```

## 🎨 메뉴별 기능 설명

### 📚 내 수업
강사의 수업 관련 모든 기능을 통합
- **수업 목록**: 담당하는 모든 수업 보기
- **주간 시간표**: 이번 주 수업 일정 확인
- **출석 체크**: 출석 체크 페이지로 이동
- **수업 공지/과제**: 수업별 공지사항 및 과제 관리

### 👥 학생 관리
담당 학생 관련 기능
- **내 학생 목록**: 담당 학생들 조회
- **피드백 작성**: 학생 피드백 작성 페이지
- **독서 논술 MBTI**: 학생 성향 분석 및 관리

### ✍️ 과제 관리
과제 및 첨삭 관련 기능
- **과제 출제/첨삭**: 과제 생성 및 첨삭하기
- **첨삭 관리**: 과제 목록 및 첨삭 관리

### 📁 학습 자료
학습 자료 관리 (단일 메뉴)

### 💬 소통
커뮤니케이션 기능
- **공지사항**: 전체 공지사항 확인
- **클래스 게시판**: 클래스별 게시판
- **강사 게시판**: 강사 전용 게시판
- **하크니스 게시판**: 하크니스 토론 게시판

## 🎯 사용자 가이드

### 메뉴 사용법
1. **카테고리 클릭**: 하위 메뉴 열기/닫기
2. **하위 메뉴 클릭**: 해당 페이지로 이동
3. **기본 상태**: "내 수업"은 항상 열린 상태로 시작

### 특징
- "내 수업"은 가장 자주 사용하는 카테고리로 기본 열림
- 여러 카테고리를 동시에 열 수 있음
- 하위 메뉴 호버 시 오른쪽으로 살짝 이동 + 색상 변경
