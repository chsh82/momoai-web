## 🎯 사용자 시나리오 비교

### 시나리오 1: "줌 접속 로그를 확인하고 싶다"

**BEFORE**:
```
1. 관리자 메뉴 스크롤 ↓↓↓
2. "온라인 강의실" 찾기
3. 클릭
4. 페이지에서 "접속 로그" 버튼 찾기
5. 클릭
```
**총 5단계**, 스크롤 필요

**AFTER**:
```
1. "🎥 온라인 강의실" 클릭 (펼쳐짐)
2. "접속 로그" 클릭
```
**총 2단계**, 스크롤 불필요
