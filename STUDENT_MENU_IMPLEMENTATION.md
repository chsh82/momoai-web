## 🎨 메뉴별 기능 설명

### 📚 내 수업
학생의 수업 관련 모든 기능을 통합
- **수업 목록**: 내가 수강하는 모든 수업 보기
- **출결 현황**: 출석 통계 및 이력 확인
- **클래스 게시판**: 같은 수업을 듣는 친구들과 소통

### ✍️ 학습 활동
학습 콘텐츠 및 과제 관련 기능 (가장 많은 하위 메뉴)
- **과제 제출**: 새로운 과제 제출하기
- **내 첨삭**: 제출한 과제의 첨삭 결과 보기
- **학습 교재**: 다운로드 가능한 학습 교재
- **학습 동영상**: 강의 동영상 시청
- **독서 논술 MBTI**: 독서 성향 분석 및 추천
- **학습 진도**: 전체 학습 진행 상황 확인

### 🔄 보강수업
보강수업 신청 및 관리
- **신청하기**: 새로운 보강수업 신청
- **신청 내역**: 과거 신청 이력 및 승인 상태

### 💬 소통
커뮤니케이션 기능
- **공지사항**: 학원 전체 공지사항 확인
- **하크니스 게시판**: 하크니스 토론 게시판

## 🎯 사용자 가이드

### 메뉴 사용법
1. **카테고리 클릭**: 하위 메뉴 열기/닫기
2. **하위 메뉴 클릭**: 해당 페이지로 이동
3. **기본 상태**: "내 수업"은 항상 열린 상태로 시작

### 특징
- "내 수업"은 가장 자주 사용하는 카테고리로 기본 열림
- 여러 카테고리를 동시에 열 수 있음
- 하위 메뉴 호버 시 오른쪽으로 살짝 이동 + 색상 변경
